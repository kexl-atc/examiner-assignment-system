## 问题分析

按钮显示"从考官分配导入 (无数据)"，说明系统没有检测到考官分配数据。

### 根本原因

1. `assignment-import-fixer.js` 检查的是 `assignment_data_for_schedule` localStorage 键
2. `data-sync-fixer.js` 只同步数据到 `teachers` 和 `students` 键
3. 没有代码将考官分配数据保存到 `assignment_data_for_schedule` 键

### 修复方案

修改 `data-sync-fixer.js`，在同步数据时同时保存到 `assignment_data_for_schedule` 键：

1. 在 `DataSyncer.syncStudentData()` 方法中，添加保存到 `assignment_data_for_schedule` 的逻辑
2. 确保数据格式符合预期：`{ students: [...], timestamp: ..., version: '1.0' }`
3. 触发 `assignmentDataUpdated` 事件通知页面更新

### 需要修改的文件

- `deploy/win7-package/frontend/assets/js/data-sync-fixer.js`

### 修复代码

在 `syncStudentData` 方法中添加：
```javascript
// 同时保存到 assignment_data_for_schedule 供排班页面导入
const assignmentData = {
    students: students,
    timestamp: Date.now(),
    version: '1.0'
};
DataManager.save(STORAGE_KEYS.ASSIGNMENT_DATA, assignmentData);

// 触发事件通知
window.dispatchEvent(new CustomEvent('assignmentDataUpdated', {
    detail: { count: students.length }
}));
```
