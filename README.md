# 智能考官排班系统

> 基于OptaPlanner的考试自动排班系统 - 完整离线部署版

## 🚀 快速开始（3步部署）

### 步骤1：解压到目标电脑

```
推荐位置：C:\examiner-system\
⚠️ 路径不要包含中文或空格
```

### 步骤2：运行智能部署

```bash
双击运行：智能部署.bat

脚本会自动：
✓ 检测Java环境（使用内置Java）
✓ 识别局域网IP
✓ 配置防火墙
✓ 启动所有服务
✓ 打开管理界面
```

### 步骤3：开始使用

```
管理控制台: http://localhost:9090/
业务系统:   http://localhost:8081/

登录信息：
用户名：admin
密码：000000
```

## 🔧 常见问题

### Q: Windows 7 可以使用吗？

**A:** ✅ 完全支持

- JDK 17.0.16 支持 Windows 7 SP1
- 所有功能正常运行
- 详见：[Windows7兼容性说明.md](docs/Windows7兼容性说明.md)

### Q: 需要安装Java吗？

**A:** ❌ 不需要

- 已包含JDK 17.0.16便携版
- 智能部署.bat会自动使用
- 无需手动安装

### Q: 局域网其他电脑如何访问？

**A:** 非常简单

```
1. 记下服务器IP（部署完成时会显示）
2. 其他电脑打开浏览器
3. 输入：http://服务器IP:8081/
4. 直接使用，无需安装
```

### Q: 如何停止服务？

**A:** 两种方式

```
方式1：管理界面 > 点击"停止"按钮
方式2：任务管理器 > 结束 supervisor-headless.exe
```

### Q: 可以多台电脑同时访问吗？

**A:** ✅ 可以

- 支持50-100人并发
- 局域网内任何设备
- 电脑、手机、平板都可以
