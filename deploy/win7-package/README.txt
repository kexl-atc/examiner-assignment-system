================================================
  快速开始
================================================

1. 解压到目标电脑（推荐路径不要包含中文或空格）

2. 双击运行 "启动服务-Win7完整版.bat"
   或双击 "快速启动-Win7完整版.vbs"（无窗口模式）

3. 等待服务启动完成（约30秒）

4. 浏览器会自动打开 http://127.0.0.1:8081

5. 使用以下信息登录管理控制台:
   - 地址: http://127.0.0.1:9090
   - 用户名: admin
   - 密码: 000000

================================================
  系统要求
================================================

最低配置:
  - 操作系统: Windows 7 SP1 (64位)
  - 内存: 2GB
  - 磁盘: 500MB 可用空间
  - Java: JDK 17 (已包含)

推荐配置:
  - 操作系统: Windows 10/11 (64位)
  - 内存: 4GB 或以上
  - 磁盘: 1GB 可用空间
  - Java: JDK 17 LTS

================================================
  故障排除
================================================

问题1: 端口被占用
  解决: 运行 "停止服务.bat" 后重试

问题2: Java 未找到
  解决: 确保 java-runtime 目录存在

问题3: 服务启动失败
  解决: 查看 logs\backend.log 和 logs\supervisor.log

问题4: 无法访问管理控制台
  解决: 检查防火墙设置，确保端口 8081, 8082, 9090 开放
