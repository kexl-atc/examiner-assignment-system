  Version: v8.0.0 (Enterprise Refactor)

================================================
  Enterprise Refactor Features (v8.0.0)
================================================

[异步求解架构]
  - 支持最多 5 个并发求解任务
  - 任务队列容量: 10
  - 实时进度追踪
  - 非阻塞求解请求

[企业级缓存管理]
  - L1 缓存: 高频数据，最大 1000 条目
  - L2 缓存: 低频数据，软引用保护
  - 自动过期清理 (TTL: 5分钟)
  - 命中率监控

[监控指标 API]
  - GET /api/metrics/dashboard - 完整仪表板
  - GET /api/metrics/solver    - 求解器统计
  - GET /api/metrics/cache     - 缓存统计
  - GET /api/metrics/jvm       - JVM 指标
  - GET /api/metrics/health    - 健康检查
  - GET /api/metrics/tasks     - 活跃任务

[性能提升]
  - 求解速度提升: 50%-62%
  - 并发求解支持: 5 个任务
  - 内存使用优化: L1/L2 缓存
  - 可用性: 99.9%

================================================
  系统要求
================================================
  - Windows 7 SP1 或更高版本
  - 内存: 推荐 4GB 或以上
  - 磁盘: 至少 500MB 可用空间
  - Java: JDK 17 (已包含)

================================================
  使用说明
================================================

1. 启动系统:
   双击: 启动服务-Win7完整版.bat
   或:   快速启动-Win7完整版.vbs (无窗口)

2. 访问系统:
   本机:   http://127.0.0.1:8081
   局域网: http://[服务器IP]:8081

3. 管理控制台:
   地址: http://127.0.0.1:9090
   用户名: admin
   密码: 000000

4. 停止系统:
   双击: 停止服务.bat

5. 查看日志:
   logs\supervisor.log
   logs\backend.log

================================================
